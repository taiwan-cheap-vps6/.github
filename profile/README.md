## 台湾 VPS 使用前必须知道的几件事

**一、非大陆直连优化，但未必是坏事**

丽萨主机台湾 VPS 的网络属于国际 BGP，不是专门面向大陆优化的。联通和部分移动用户直连效果还不错，但电信用户直连可能有波动，建议先用香港或日本节点中转。

**二、不限流量 VDS 的退款政策不同**

有限流量的 VPS 套餐（包括年付版）享有 48 小时无条件退款，**不限流量的 VDS 产品退款只退网站余额**，无法原路退还，购买前请确认好需求。

为防止 SSH 扫爆攻击，新开机的服务器 SSH 端口不是默认的 22，需要在管理面板里查看实际端口，不要以为连不上就是故障。

这条可能有人没注意——台湾 VPS 系列全线支持安装 Windows，对于需要跑 Windows 程序或 GUI 界面的用户是个加分项。
